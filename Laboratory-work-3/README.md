## \## Висновок

## 

## У ході виконання лабораторної роботи було отримано практичні навички роботи з оболонкою Bash. Було розглянуто навігацію файловою системою, перегляд прихованих файлів, роботу з каталогами та конфігураційними файлами Git. Також було створено псевдоніми команд Bash і налаштовано їх автоматичне завантаження через файл `.bash\_profile`.

## 

## Під час роботи було використано перенаправлення потоків введення-виведення для створення та доповнення текстових файлів. Крім того, було застосовано команди `cat`, `paste`, `tr`, `sort`, `head` та `wc` для простої обробки текстових даних. Окремо були закріплені навички роботи з Git та GitHub: створення гілки, виконання комітів і відправлення змін до віддаленого репозиторію.
