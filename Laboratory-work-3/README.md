## \# Висновок

## 

## У ході виконання лабораторної роботи було отримано практичні навички роботи з оболонкою Bash. Було освоєно навігацію файловою системою, використання псевдонімів команд, роботу з файлами за допомогою перенаправлення потоків введення-виведення, а також застосування конвеєрів команд для обробки текстових даних. Крім того, були закріплені навички роботи з Git та GitHub, створення окремих гілок репозиторію, виконання комітів і синхронізації локального репозиторію з віддаленим.
